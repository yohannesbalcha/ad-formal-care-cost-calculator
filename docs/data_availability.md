# Data Availability

Individual-level data used to develop the cost function are not included in this repository.

The study used linked Swedish registry data. These data are subject to legal, ethical, and data-sharing restrictions and cannot be redistributed publicly. Access to the source data requires appropriate approvals from the relevant Swedish registry holders and ethics authorities.

This public repository includes only:

- coefficient tables required to implement the published cost function;
- a Shiny calculator using those coefficients;
- variable definitions;
- synthetic example input rows.
